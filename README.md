# MSP-1 Schemas

This repository contains the canonical JSON Schema definitions for the MSP-1 (Mark Semantic Protocol).

These schemas define the machine-readable structure for:
- Site-level MSP-1 declarations
- Page-level MSP-1 declarations

This repository is the source of truth for schema validation.
Narrative documentation lives at https://msp-1.org
